## 组命令和子 shell

组命令：

{ command1; command2; [command3; ...] }
Subshell:

子 shell：

(command1; command2; [command3;...])

> 使用{}要特别的注意要有一个空格，[]其实也是要求有空格，但是（）并不需要

它们都是用来管理重定向的
