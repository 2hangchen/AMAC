# 原型模式

```go
type Pro interface {
	Get()
}
type prototype struct {
	Value map[string]Pro
}

func(p *prototype)Get(s string)Pro{
	return p.Value[s]
}
func(p *prototype)Set(s string,pr Pro){
	p.Value[s]= pr
}
func NewPrototype()*prototype {
	return &prototype{
		Value: make(map[string]Pro),
	}
}
```
原型模式就是这种使用了set一个方法进入，然后用get取出的这种方式的模式。
