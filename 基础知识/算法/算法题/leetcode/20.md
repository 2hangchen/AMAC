#  Valid Parentheses 有效的括号
## 解法一
```go
func isValid(s string) bool {
	// 设置一个辅助的map
	aid := make(map[rune]rune)
	aid['('], aid['{'], aid['['] = ')','}',']'
	// 设置一个辅助的stack
	stack := godata.NewStack(6) //github.com/googege/godata  
	for _, v := range s {
		if _, ok := aid[value]; ok { // 存在也就是左边
			stack.Push(value)
		} else { //不存在也就是右边
			top := stack.Pop()
			if top != nil { // 右边多
				if value != aid[top.(rune)] {
					return false
				}
			}else {
				return false
			}
		}
	}
	if stack.Length() > 0 {
 		return false // 左边多
	}
	return true
}
```

## 解法二

在算法题中假如你的编程语言没有基本的数据结构比如stack 自己实现的时候只实现核心部分即可

这个题 只实现的 一个slice即可

```go
func isValid(s string) bool {
    m := make(map[rune]rune)// 辅助map
    m['('],m['{'],m['['] = ')','}',']'
    st := make([]rune,0,len(s))
   for _,v := range s {
        if _,ok := m[v];ok {
            st = append(st,v)
        }else {
            if len(st) == 0 { // 假如没有 append就证明是错的。
                return false
            }
            top := st[len(st)-1]
            if v != m[top]||len(st) ==0{
                return false
            }
            if len(st) >0 {
              st = st[:len(st)-1]   
            }
        }
    }
    if len(st) > 0 {
        return false
    }
    return true
}
```
