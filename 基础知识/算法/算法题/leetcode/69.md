# sqrtx
## 解法一
binary search 二分查找

```go
func mySqrt(x int) int {
    if x < 2{
        return x
    }
    a , b := 0 , x //设置两个值，表示最大值和最小值
    for b - a > 1{ //限定条件
        c := (a + b)/2  //二分查找
        if (x/c < c){  //用x/c <c 比c*c >x好，可防止溢出
            b = c
        }else{
            a = c
        }
    }
    return a
}

```
