# climbing stairs
爬楼梯问题其实就是斐波那契数列
## 解法一
动态规划
```go
func climbStairs(n int) int {
    n++
    ma := make([]int,n+1)
    ma[0] = 0
    ma[1] = 1
    for i:= 2;i <=n;i++ {
        ma[i] = ma[i-1]+ma[i-2]
    }
    return ma[n]
}
```
