# Linked List Cycle 寻找链表的环
## 解法一 快慢指针

```go

func hasCycle(head *ListNode) bool {
    fast,slow := head,head
    for fast != nil && slow != nil && fast.Next != nil { // 设置跳出值
        fast = fast.Next.Next // 快指针通常是慢指针的一个倍数
        slow = slow.Next
        if fast == slow {
            return true
        }
    }
    return false
}
```
