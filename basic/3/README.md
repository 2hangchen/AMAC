## 队列

队列跟栈非常相似，也是 进 出 这两个操作，但是它是 FIFO 也就是先进先出 而 对比来说 栈是 先进后出 举个例子 ：

一个米缸 放小鱼干 然后从下面开一个孔然后把小鱼干抽出来的过程 就是队列。就是 谁最第一个先进去，谁就第一个先出来。

```go
//队列：FIFO 先进先出
//进去的顺序是 1 2 3 4 5
// 出来的顺序是 1 2 3 4 5

//  栈:FILO 先进后出
// 进去的顺序是 1 2 3 4 5
// 出来的顺序是 5 4 3 2 1
```

### 我们可以实现的队列的形式有一下几种

- 顺序队列

- 循环队列

- 阻塞队列

- 并发队列
    - 锁
    - 原子操作
