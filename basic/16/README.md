# 动态规划

动态规划，英文是：Dynamic Programming. 动态规划是和递归一样，是算法思想中最重要的思想之一，是跟递归一样重要甚至更重要的算法思想理念。
就跟很多著名的算法一样，例如跳表，动态规划也是一种用空间换时间的方法。
